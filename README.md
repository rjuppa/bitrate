# bitrate
Get price of 1BTC

Install
=======
$ git clone git@github.com:rjuppa/bitrate.git
$ cd bitrate
$ virtualenv venv
New python executable in venv/bin/python
Installing setuptools, pip............done.
$ . venv/bin/activate
$ pip install --editable .
.....

$ bitrate
According to BitStamp.net the price for 1BTC is:
292.55 USD
